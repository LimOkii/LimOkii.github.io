# Yifei Lu Academic Homepage

Static personal academic homepage for GitHub Pages.

## Local Preview

Open `index.html` directly in a browser, or run:

```powershell
python main.py --port 8000
```

Then visit `http://127.0.0.1:8000`.

## Publish to GitHub Pages

1. Create a public GitHub repository named `LimOkii.github.io` for a user homepage, or any repository name for a project page.
2. Push this directory to the repository's `main` branch.
3. In GitHub, open `Settings -> Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Select branch `main` and folder `/ (root)`.
6. Save and wait for GitHub Pages to publish.

For a user homepage, the site URL will be:

```text
https://LimOkii.github.io/
```

For a project page, the URL will be:

```text
https://LimOkii.github.io/<repository-name>/
```
